    Getting Started¶

    This section walks you through installing Scrub and running your first batch export.

    Prerequisites¶

    Before installing Scrub, make sure you have:

    • Adobe Premiere Pro 25.6 or later installed
    • UXP Developer Tools (UDT) v2.2+ — download from the Adobe Developer Console
    • A supported operating system: Windows 10/11 or macOS 13+
